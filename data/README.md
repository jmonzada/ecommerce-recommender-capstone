# Data

## Source

**Brazilian E-Commerce Public Dataset by Olist** - https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

Real, anonymized commercial data: ~100k orders placed on the Olist marketplace between 2016 and 2018, across 9 relational CSV files (orders, order items, customers, products, sellers, reviews, payments, geolocation, and a category-name translation table).

## Download (to regenerate `data/raw/`)

```bash
pip install kaggle
# place your Kaggle API token at %USERPROFILE%\.kaggle\kaggle.json first
kaggle datasets download -d olistbr/brazilian-ecommerce -p data/raw --unzip
```

Or download manually from the Kaggle dataset page and extract the CSVs into `data/raw/`.

## Rules

- Files in `data/raw/` are never edited, renamed, or re-encoded. All transformations write derived artifacts to `data/processed/` (gitignored; regenerated by the notebooks/`src` pipeline).
